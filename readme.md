uSchedule
=========

Installation:
-------------
```
1. Console: "composer install"
2. Console: "npm install"
3. Clone .env.example to .env file
4. Config .env file for your configuration
5. Console: "php artisan key:generate"
6. Console: "php artisan jwt:secret"
.. @todo Frontend
```
